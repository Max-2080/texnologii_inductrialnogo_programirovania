приложение создано для добавления задач и дел на неделю. пользователь вводит в первую строку название задачи, например можно написать СДЕЛАТЬ ЗАРЯДКУ, 
далее можно ввести в соответствующее поле описнаие задачи, которое напримре  может понадобиться пользователю, затем из выпадающего списка пользователь 
может выбрать день недели, в который необходимо выполнить задачу, а также приоритетность, которая может быть средней, низкой или высокой. Для удобства 
просмотра задач я предусмотерл возможность фильтрации задач по дням недели, нажав на какой-то день, пользователь увидит список задач для этого дня
